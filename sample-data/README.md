# Sample data

Five real AiM `.xrk` logs from a **24 Hours of Lemons** race at **NJMP Thunderbolt** (Sunday, one race day, four driver stints, ~185 laps total). Recorded on a Solo-type GPS logger on a budget Honda.

| File | Stint (driver) | Approx. time (EDT) | Laps | Best lap |
|------|----------------|--------------------|------|----------|
| `..._0418.xrk` | Driver 1 | 08:54–10:02 | 35 | 1:46.794 |
| `..._0421.xrk` | Driver 1 | 10:24–10:58 | 18 | 1:49.589 |
| `..._0423.xrk` | Driver 2 | 11:03–12:28 | 42 | 1:53.021 |
| `..._0425.xrk` | Driver 3 | 12:32–14:01 | 45 | 1:52.080 |
| `..._0426.xrk` | Driver 4 | 14:05–15:35 | 45 | 1:52.505 |

To try the analyzer: open [`../index.html`](../index.html) and drag all five files in. Driver names default to "Driver 1–4" in file order; in the Sessions panel you can rename them, and the first two files share the name "Driver 1" because they were the same driver's two stints.

These files are shared purely as a demo/test fixture. No personal identifying data is stored in them (the racer/vehicle fields are blank).
